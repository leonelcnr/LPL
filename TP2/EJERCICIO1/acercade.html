ACERCA DE PAGINAS WEBS

  • Las páginas web pueden ser estáticas o dinámicas.
  • Las páginas web estáticas fueron las primeras en desarrollarse.
  • Aún se utilizan en la actualidad.
  • Brindan soporte para la presentación de datos en diversas situaciones.
  • Son fáciles de crear, instalar y mantener.
  • No permiten una gran interacción con el usuario.
  • Para mayor interacción, se necesitan páginas web dinámicas.
  • Las páginas dinámicas permiten procesar datos ingresados por el usuario (por ejemplo, en formularios).
